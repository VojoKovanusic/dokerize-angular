Java Core

Radi se o kursu programiranja Jave. Trajanje kursa 5 mjeseci (2-3 puta sedmično, Teslić). Predavač Java programer sa iskustvom. Plan i program uskaldjen sa potrebama tržišta. Ciljna grupa završne godine srednje škole, nezaposlene osobe do 29 godina starosti. Organizator NVO Razvojni centar Teslić. Akcenta nije samo na teoriskom predavanju, već i u interakciji sa predavačem, ostalim polaznicima uz mnogo logičkih zadataka koji će biti odličan temelj za razvoj IT karijere. Cjena kursa zavisi od broja polaznika, okvirno izmedju 50-100 km / mjesečno.
